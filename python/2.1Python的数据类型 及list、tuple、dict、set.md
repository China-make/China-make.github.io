4、字典{}
字典是python中惟一的一个映射类型（哈希表）
字典对象是可变的，但是字典的键必须是不可变的对象，并且一个字典中可以使用不同类型的键值。

dict与list的比较：
dict:查找和插入速度极快，不会随着key的增加而增加；
需要占用大量的内存，内存浪费多
key不可变
默认无序
list：查找和插入的时间随着元素的增加而增加；
占用空间小，浪费内存很少
通过下标查询
有序
------------------------------------------------------

{ }

取字典中的一个值，当它不存在时会返回一个自定义的一个信息。如：dic.get(5,'error')
